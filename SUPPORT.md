# Support

Fractals is an experimental, solo-maintained project. There is no guaranteed support response time.

Use GitHub issues for reproducible bugs, documentation corrections, and scoped proposals. Include the version or commit, operating system, browser, OpenCode version, command run, expected behavior, actual behavior, and any relevant logs with secrets and private project data removed.

Questions that depend on private projects, account setup, unrelated consulting, or custom OpenCode forks may be closed or redirected.
